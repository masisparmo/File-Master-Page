# File Master Landing Page 🚀✨

A premium, bilingual (ID/EN) promotional landing page for **File Master**, a powerful AI-driven file manager for Windows.

## Features Showcase
- 🌐 **Bilingual Support**: Full Indonesian (default) and English translations.
- 🌓 **Mode Dark/Light**: Seamless theme switching with persistent user preference.
- 🔍 **Interactive Lightbox**: High-resolution image zoom for application screenshots.
- 📱 **Fully Responsive**: Optimized for Desktop, Tablet, and Mobile viewports.
- 🤖 **AI-Driven Copywriting**: Uses "Loss Aversion" strategy to engage users.
- 🎨 **Premium Aesthetics**: Modern glassmorphism, smooth animations, and optimized tipography.

## Tech Stack
- **HTML5 & Vanilla CSS**: Clean, semantic structure with custom CSS variables.
- **Vanilla JavaScript**: Dynamic interactions and translation engine.
- **SEO Optimized**: Complete with meta tags, Open Graph (OG), and accessibility features.

## How to View
1. Clone this repository.
2. Open `index.html` in your browser.
3. Or run a local server: `python -m http.server 8000`.
